# ⚡ TECH TUTORIALS – Interactive Tech Learning Platform

Welcome to **TECH TUTORIALS**, a fully functional, PWA‑enabled website built to deliver tech guides, tools, and tutorials. It features a mandatory email sign‑up gate (powered by MailerLite), password‑protected video content, progress tracking, and a user dashboard—all running entirely on static HTML, CSS, and JavaScript with offline support.

---

## 🚀 Features

- **📱 Progressive Web App (PWA)** – Install directly to your device’s home screen (Android, iOS, Desktop).
- **🌗 Dark / Light Mode** – Toggle themes with persistence via `localStorage`.
- **🔍 Live Search** – Filter tutorial cards on the homepage instantly.
- **🔐 Video Password Protection** – Watch videos only after entering the correct password (`TECH11`).
- **⛔ Rate Limiting** – 3 incorrect attempts = 5‑minute block to prevent brute‑forcing.
- **🎬 Lazy‑Loaded Videos** – Iframes load *only* after successful password entry (faster initial load).
- **✅ Progress Tracking** – Mark videos as “Watched” (saved locally per device).
- **⏪⏩ Prev / Next Navigation** – Smooth scrolling between tutorial methods.
- **📬 Mandatory Sign‑Up Gate** – Users must subscribe via MailerLite before accessing content.
- **📊 User Dashboard** – View subscription status, email, and clear local data.
- **🚫 Video Download Protection** – Right‑click blocked on video containers to prevent direct downloading.
- **🧭 Breadcrumbs** – Clear navigation hierarchy on every page.
- **📄 Custom 404 Page** – Suggests alternative tutorials when a page is not found.
- **📱 Fully Responsive** – Works beautifully on mobile, tablet, and desktop.
